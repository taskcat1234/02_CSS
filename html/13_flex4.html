flex를 이용한 화면(영역) 분할

상하 2분할(flex 사용 한해도 가능)

    width  : 500px;
    height : 600px;
    영역을 위/아래로 절반씩 나누기
    -> display : block; 특징 활용

세로 방향 3분할

    width  :  300px
    height :  432px
    1행 : 2행 : 3행 = 50 : 30 : 20
    (각 행의 배경색은 자유)

좌우 2분할

    width  : 400px;
    height : 200px;
    좌우 절반씩 영역 분할

가로 5분할

    width  : 600px;
    height : 100px;

    1,3,5 : 너비 10%
    2,4   : 너비 35%
